# No Fog – Clear View Resource Pack

### Clear your world. See Minecraft like never before.

---

## Overview
**No Fog – Clear View** is a lightweight **Minecraft resource pack** that completely removes the annoying **fog effect** from your game — allowing you to see **crystal-clear distances** across all biomes, dimensions, and weather conditions.  

Perfect for players who love exploration, building cinematic landscapes, or simply want a cleaner and sharper visual experience.

---

## Features
- Removes fog in all biomes and dimensions (Overworld, Nether, End)
- Improves visibility even during rain, snow, or storms
- No performance loss — lightweight and optimized
- Compatible with shaders & OptiFine / Iris
- Works for both Survival and Creative modes

## Installation
1. Download the `.zip` file of the pack.  
2. Launch **Minecraft** → go to **Options → Resource Packs**.  
3. Click **“Open Pack Folder”** and move the file inside.  
4. Activate **No Fog – Clear View** in your resource packs list.  
5. Enjoy your fog-free experience!
